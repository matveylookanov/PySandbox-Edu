# PySandbox Edu

Учебный проект — онлайн-песочница для безопасного запуска Python-кода в изолированных Docker-контейнерах.

## Команда разработки
* **ФИО 1** — Frontend Developer / Team Lead
* **ФИО 2** — Backend Developer
* **ФИО 3** — DevOps / QA Engineer

## Технологический стек и архитектура
Проект построен на микросервисной архитектуре:
*   **Frontend**: Nginx + Static Web (HTML/JS), порт `3000`.
*   **Backend**: FastAPI (Python 3.10), управление жизненным циклом контейнеров через Docker SDK, порт `8000`.
*   **Executor**: Изолированный легковесный образ на базе Python 3.10 для выполнения пользовательских скриптов.

### Механизмы изоляции
Запуск кода производится в эфемерных контейнерах со следующими ограничениями:
*   **Memory Limit**: 128 MB.
*   **CPU Limit**: 0.5 vCPU.
*   **Networking**: Полное отсутствие сетевого доступа (`network_disabled=True`).
*   **Auto-Cleanup**: Автоматическое удаление контейнера после завершения процесса (`auto_remove=True`).

Подробное описание представлено в [docs/Архитектура.md](docs/Архитектура.md).
